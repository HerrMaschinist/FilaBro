## Schritt 4: Farbe Normalisierung in Application Layer
Erstelle /src/core/application/filament/ColorNormalizer.ts

Regeln:
- Input string trim, lower
- Wenn Hex Pattern, normalisiere auf #RRGGBB und setze colorHexNormalized
- Wenn Text, mappe über Alias Tabelle:
  - schwarz, black -> Black + #000000
  - weiß, weiss, white -> White + #FFFFFF
  - grau, gray, grey -> Gray + #808080
  - rot, red -> Red + #FF0000
  - grün, green -> Green + #00FF00
  - blau, blue -> Blue + #0000FF
  - gelb, yellow -> Yellow + #FFFF00
  - orange -> Orange + #FFA500
  - lila, purple -> Purple + #800080
  - pink -> Pink + #FFC0CB
  - braun, brown -> Brown + #8B4513
- Wenn kein Match: setze colorNameRaw = original, normalized und hex bleiben undefined

Update /src/core/application/FilamentUseCase.ts
- updateFilament(localId, patch):
  - wenn patch.colorInput gesetzt, normalisiere und setze Felder
  - wenn patch.spec gesetzt, übernehme Werte
  - ruft repo.updateLocal mit finalem Patch
